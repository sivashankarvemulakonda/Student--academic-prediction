# Student Performance Analytics Platform

A machine learning application that predicts student academic performance based on electronic gadget usage patterns.

## Features
- Predict student performance (High/Medium/Low)
- Analyze gadget usage patterns
- Interactive frontend with real-time predictions
- ML-powered recommendations

## Project Structure
- `backend/`: Flask API server
- `frontend/`: Web interface
- `paper.ipynb`: Research and analysis notebook

## Getting Started
1. Deploy backend to Render
2. Serve frontend files
3. Access the web interface

## API Endpoints
- `POST /predict`: Make predictions
- `GET /health`: Health check
- `GET /features`: Get feature information
